# Boilermaker

This is boilermaker! This project allows you to get up an running with  NODE, EXPRESS, REACT, REDUX, SEQUELIZE (NERDS STACK).

## Getting Started

To get started:

Please make sure you have nodemon installed globally.

```
npm install nodemon -g

```

### Prerequisites


```
npm install
npm run start
```
